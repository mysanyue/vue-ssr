# vue-ssr
vue 同构应用，自动部署

**提交发布**

1. `git add .`
2. `git commit -m` "提交部署"
3. `git push -u origin master`
4. `git tag v0.1.0` 打标签
5. `git tag`  列出标签
6. `git push origin v0.1.0`

### DEOME 地址

[https://demo.realworld.io/#](https://demo.realworld.io/#)

### 要求

- 首先到 deom 地址注册一个自己的账号，方便自己代码测试

- 实现功能包括：
  - [x]注册
  - [x]登录
  - [x]退出
  - [x]文章发表
  - [x]修改
  - [x]列表
  - [x]分页
  - [x]标签
  - [x]详情
  - [x]点赞/取消点赞
  - [x]个人中心
  - [x]个人信息更改
